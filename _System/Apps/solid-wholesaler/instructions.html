Solid Meta Tags - Version 1.0

Description

If any of the following guidelines are not met for a page, they will show up as issues and the row background will be red:

  1. Page Title exists and is between 8 and 70 characters (ideally 55 characters)
  2. Meta Description exists and is less than 160 characters
  3. URL is less than 70 characters
  4. URL should not contain a file extension, expect for index files (ex .html)

This app only scans live pages on the website, that are not excluded from showing up on the search engines.
